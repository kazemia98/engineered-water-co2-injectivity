# Data description

This folder contains the processed dataset used to generate all results
presented in the manuscript.

## Files

- `engineered_water_screening_results.csv`  
  Master dataset containing:
  - engineered-water composition multipliers
  - conditioning cost proxy (CostIndex)
  - storage readiness proxy (DIC at 1000 years)
  - early injectivity risk metrics (0–10 y and 0–100 y)
  - Pareto ranking indicators
  - Sustainability Index (SI_SUS)

All figures and tables in the manuscript are derived from this file.
